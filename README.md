# Internship Task - Authentication Pages

## Overview
This project implements the **Register, Login, and Forgot Password pages** as per the internship assignment.  
It is built using **React**, **Vite**, **Tailwind CSS**, and integrates with the provided **API endpoints**.

---

## Features
- **Login Page**: Email & password authentication with API integration.  
- **Register Page**: Business & personal details, phone number, email, password, terms & conditions, captcha verification.  
- **Forgot Password Page**: Send OTP, reset password with API integration.  
- Fully responsive and styled with **Tailwind CSS**.  

---

## Tech Stack
- React (JSX, Hooks)
- Vite (Build Tool)
- Tailwind CSS (Styling)
- PostCSS & Autoprefixer
- React Router (Routing)
- Axios (API calls)
